## 📖 Описание базы данных

База данных создавалась в рамках учебного курса по MySQL и включает в себя основные сущности, необходимые для работы современного мессенджера:

- **Пользователи** — регистрация, профили, настройки.
- **Каналы** — публичные/приватные каналы для трансляций.
- **Группы** — чаты с несколькими участниками.
- **Сообщения** — текстовые сообщения с медиавложениями.
- **Реакции** — лайки и другие реакции на сообщения.
- **Настройки** — пользовательские настройки (тема, язык, уведомления).

## 📂 Структура базы данных

### 👤 Пользователи (`users`)

Хранит информацию о зарегистрированных пользователях.

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный идентификатор пользователя |
| `firstname` | VARCHAR(50) | Имя пользователя |
| `lastname` | VARCHAR(50) | Фамилия пользователя |
| `email` | VARCHAR(100) | Email (уникальный) |
| `password_hash` | VARCHAR(255) | Хеш пароля |
| `phone` | VARCHAR(20) | Номер телефона |
| `birthday` | DATE | Дата рождения |
| `created_at` | DATETIME | Дата регистрации |
| `is_premium_account` | BOOLEAN | Флаг премиум-аккаунта |

---

### 📢 Каналы (`channels`)

Публичные или приватные каналы для вещания контента.

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный идентификатор канала |
| `title` | VARCHAR(100) | Название канала |
| `owner_user_id` | INT (FK) | Создатель канала |
| `is_private` | BOOLEAN | Флаг приватности |
| `invite_link` | VARCHAR(255) | Ссылка-приглашение |
| `created_at` | DATETIME | Дата создания |

---

### 💬 Группы (`groups`)

Чаты с несколькими участниками.

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный идентификатор группы |
| `title` | VARCHAR(100) | Название группы |
| `created_at` | DATETIME | Дата создания |

---

### 👥 Участники групп (`group_members`)

Связь между пользователями и группами (многие-ко-многим).

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `group_id` | INT (FK) | ID группы |
| `user_id` | INT (FK) | ID пользователя |
| `created_at` | DATETIME | Дата присоединения |

---

### 📨 Сообщения

#### Каналы — `channel_messages`

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный ID сообщения |
| `channel_id` | INT (FK) | ID канала |
| `sender_id` | INT (FK) | Отправитель |
| `body` | TEXT | Текст сообщения |
| `media_type` | VARCHAR(20) | Тип вложения (image, video, etc.) |
| `filename` | VARCHAR(255) | Имя файла |
| `views_count` | INT | Количество просмотров |
| `code` | TEXT | Код (если это код) |
| `created_at` | DATETIME | Дата отправки |

#### Группы — `group_messages`

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный ID сообщения |
| `group_id` | INT (FK) | ID группы |
| `sender_id` | INT (FK) | Отправитель |
| `body` | TEXT | Текст сообщения |
| `media_type` | VARCHAR(20) | Тип вложения |
| `filename` | VARCHAR(255) | Имя файла |
| `reply_to_id` | INT (FK) | Ответ на другое сообщение |
| `created_at` | DATETIME | Дата отправки |

#### Личные — `private_messages`

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный ID сообщения |
| `sender_id` | INT (FK) | Отправитель |
| `receiver_id` | INT (FK) | Получатель |
| `body` | TEXT | Текст сообщения |
| `media_type` | VARCHAR(20) | Тип вложения |
| `filename` | VARCHAR(255) | Имя файла |
| `reply_to_id` | INT (FK) | Ответ на сообщение |
| `is_read` | BOOLEAN | Прочитано/не прочитано |
| `created_at` | DATETIME | Дата отправки |

### ⚙️ Настройки (`settings`)

Хранит пользовательские настройки.

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | INT (PK) | Уникальный ID |
| `user_id` | INT (FK) | Пользователь |
| `app_language` | VARCHAR(10) | Язык интерфейса |
| `color_scheme` | VARCHAR(20) | Цветовая схема |
| `is_night_mode_enabled` | BOOLEAN | Ночной режим |
| `notifications_and_sounds` | JSON | Настройки уведомлений |
| `status` | VARCHAR(50) | Статус пользователя |
| `status_text` | TEXT | Текст статуса |
| `icon` | VARCHAR(50) | Иконка статуса |
| `created_at` | DATETIME | Дата создания |
| `updated_at` | DATETIME | Дата обновления |
